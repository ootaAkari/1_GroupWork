CURRICULUM

６ヶ月間の学習内容

WSS（ウェブスタディサッポロ）では
半年間の学習をどんな流れで進めていくのか、
具体的な内容について順番にご紹介していきます。

BASIC

基礎学習

１〜2ヶ月目は基本的な言語やソフトの学習から始まります。最初に建築でいう基礎部分を作る『HTML』、内装等の装飾に例えられる『CSS』の基本を実際にコードを書いて進めます。その後、Adobe Photoshop・XDに触れていきソフトの使い方から始め、WEBサイトのレイアウトデザインを作成できるように学習します。

訓練科目

Web知識講習　/　HTML・CSS基礎実習　/　Webデザイン基礎実習　/　Webデザイン制作実習

APPLIED

応用学習

3〜4ヶ月目は、1〜2ヶ月目で学んだことを応用しグループでWEBサイトを作成します。実際の現場を想定しクライアントからの受注→デザイン→コーディングと進め、サイトを完成させます。その後、サイトに動きをつけるプログラミング言語『Javascript』、それを簡単に扱えるようにした『jQuery』を学習していきます。

訓練科目

CSS設計　/　コーポレートWebサイト実習
JS実習　/　Sass（SCSS）

PRODUCTION

制作実習

5〜6ヶ月目は、レスポンシブデザインに触れていきます。PCだけでなくスマートフォンやタブレット端末での閲覧を視野に入れたWEB制作の学習は、 WEB業界への就職にも役立つスキルです。この時期に自分の制作した作品をまとめ、『ポートフォリオサイト』を作成し就職活動を本格的に進めていくこととなります。

訓練科目

レスポンシブWebデザイン基礎実習　/　レスポンシブWebデザイン制作実習
Webサイト企画・デザイン実習　/　Webサイト制作実習
